秘钥建立

```

Client

ssh-keygen -t rsa scp /root/.ssh/id_rsa.pub root@xxx.xx.xx.xx:/root/.ssh/test_pub

Server

cat /root/.ssh/test_pub >> /root/.ssh/authorized_keys ```

安装AutoSSH

autossh下载地址：http://www.harding.motd.ca/autossh/
