setting

新建文件模板的使用

File->Setting->Edite->File and Code Templates ```

!/usr/bin/env python

@Time : ${DATE} ${TIME}

@Author : LiuWei

@Site : ${SITE}

@File : ${NAME}.py

@Software: ${PRODUCT_NAME}

- - coding: utf-8 - -

```

新建测试文件

新建Python脚步文件时点击下面的下拉框即可

与vim快捷键冲突设置

在File->Setting->Vim Emulation中对相应的快捷键选择IDE或Vim即可

自动换行设置

File->Setting->Edite->General中勾选Soft Wraps中勾选第一个和第三个即可
